# <Title scope="스타일 가이드" korean="개요" />

스타일 가이드의 구성 요소로는 색상(Color), 서체(Typography), 형태(Shape), 배치(Layout), 아이콘(Icon)이 있다.

## 적용 범위

- **정부상징 로고를 사용하는** 중앙행정기관(부/처/청), 특별지방행정기관, 부속기관 등의 웹사이트, 모바일 앱 등의 경우
  **필수로** 사용 규칙을 준수하여 일관성을 유지하도록 한다.
- **독자적인 로고를 사용하는** 중앙기관 (운영서비스·시스템) 및 공공기관 (대표·운영서비스·시스템) 등의 웹사이트, 모바일 앱 등의 겨우
  **가능한 한** 사용 규칙을 준수하도록 한다.

## 방향성

- 일관성(Consistency) : 디자인 요소를 통해 품질유지와 신뢰도를 끌어올릴 수 있도록 한다.
- 접근성(Accessibility) : 서비스 사용자에게 불편함이 없도록 [W3C 웹 콘텐츠 접근성 지침(WCAG)](https://www.w3.org/WAI/WCAG22/Understanding/contrast-minimum.html) 레벨 AA 이상의 원칙을 준수하도록 한다.
- 효율성(Efficiency) : 다양한 개발 환경을 고려해 소통 비용을 줄이고 적용이 용이한 규칙과 규격을 기반으로 한 가이드를 제시한다.
