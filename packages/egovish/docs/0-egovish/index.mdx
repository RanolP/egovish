# egovish

egovish는 행정안전부에서 마련한 [〈디지털 정부서비스 UI/UX 가이드라인〉](https://uiux.egovframe.go.kr/guide/index.html)을 준수하는 것을 목표로 삼고 있는 크로스-프레임워크 컴포넌트 라이브러리이다.
디지털 서비스 사용자에게 고품질 사용자 경험을 제공하기 위해 구현해야 할 화면 구성 요소를 최대한 제공하기 위해 노력하고 있다.

## 개발 배경

가이드라인 홈페이지에서 훌륭하게도 HTML 마크업 및 사용 가이드라인 등을 제공하고 있으나, 충분히 컴포넌트화가 이루어지지 않아 개발에 불편함이 있다.
하지만 동시에 어떠한 프레임워크로 가능성을 제한시킬 경우, 도입이 어렵다거나 기술이 정체되는 등 문제가 발생할 가능성이 높다.
따라서, 본 프로젝트는 [Builder.io, Inc.](https://www.builder.io/)에서 개발한 오픈소스 프레임워크 [mitosis](https://github.com/BuilderIO/mitosis)를 활용하여 크로스-프레임워크 라이브러리를 개발하고자 한다.

## 목적

본 라이브러리의 개발 목적은 다음과 같다.

- **첫째, 개발자의 손쉬운 사용** <br />
  말뿐인 가이드라인은 결코 성공적이라고 볼 수 없다.
  실질적인 도입이 용이하도록 편리한 라이브러리를 개발하도록 한다.
- **둘째, 변화에 대응하는 견고하고 유연한 구조** <br />
  각 구성 요소는 가이드라인의 방향성에만 부합한다면 상황에 따라 유연한 사용이 가능해야 한다.
  가능성을 제한하지 않되, 견고하고 유연한 구조를 설계하도록 한다.
